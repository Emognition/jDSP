Transforms

Provides a demonstration of using Fourier Transforms and Hilbert Transform through JDSP.

  • Fourier Transforms
  • Hilbert Transform

Discrete Fourier Transform

Signal Used for this Demo

Only Positive Values

Only Positive Complex Values

                    
DiscreteFourier fft = new DiscreteFourier(signal);
fft.dft();
boolean onlyPositive = true;
double[] out = fft.returnAbsolute(onlyPositive); //Positive Absolute
                    
                  
                    
DiscreteFourier fft = new DiscreteFourier(signal);
fft.dft();
boolean onlyPositive = true;
double[] out = fft.returnFull(onlyPositive); //Positive Complex

All Absolute Values

All Complex Values

                    
DiscreteFourier fft = new DiscreteFourier(signal);
fft.dft();
boolean onlyPositive = false;
double[] out = fft.returnAbsolute(onlyPositive); //Full Absolute
                    
                  
                    
DiscreteFourier fft = new DiscreteFourier(signal);
fft.dft();
boolean onlyPositive = false;
double[] out = fft.returnFull(onlyPositive); //Full Complex

Inverse Discrete Fourier Transform

Sequence Used for this Demo
                    
// Automatically detects real and complex sequence and performs IDFT accordingly
InverseDiscreteFourier transformer = new InverseDiscreteFourier(complexSequence);
transformer.idft();
double[] outReal = transformer.get_real_signal(); //To get only the real component
double[] outAbsolute = transformer.get_absolute_signal(); //To get the absolute value of the complex signal
double[][] out = transformer.get_complex_signal(); // To get the complex signal

Hilbert Transform

Sequence Used for this Demo
Amplitude Envelope of the Signal
                    
Hilbert h = new Hilbert(signal);
h.hilbert_transform();
double[][] analytical_signal = h.get_output(); //Returns the analytical signal
double[] envelope = h.get_amplitude_envelope();
Instantaneous Phase & Frequency of the Signal
                    
Hilbert h = new Hilbert(signal);
h.hilbert_transform();
double[] phase = h.get_instantaneous_phase();
int Fs = 150; //Sampling Frequency of the original signal
double[] frequency = h.get_instantaneous_frequency(Fs);
